# infra-lite

一个轻量级、可嵌入的 C / C++ 基础设施（Infrastructure）代码集合。

`infra-lite` 用于沉淀那些**经常被重复实现、但又不值得引入大型框架**的底层能力。

本仓库关注的不是“功能数量”，而是：

> **清晰的抽象、明确的边界、以及长期可维护性**

---

## 设计理念（Philosophy）

- 基础设施优先于功能
- 简单优先于完整
- 显式优先于隐式
- 组合优先于整体

本仓库会**缓慢、克制地增长**，每一个模块都应当：
- 足够小
- 易于理解
- 可独立复用

---

## 核心特性

- Header-first 设计
- 易于嵌入到现有工程
- 适合长期维护的代码库

---

## 构建与测试

```bash
mkdir build && cd build
cmake ..
make
ctest
```
